# N-Body Gravitational Simulation
An N-Body Gravitational simulation starting from a Plummer sphere.

The following is an implementation for the High Performance Computing exam at University of Trieste.


## Repository Structure
The repo is organized as follows:

```
├── configs/                # Configuration files for the simulation
├── docs/                   # Documentation files
├── src/                    # Source code files
└── plots/                  # Plots and images
```
